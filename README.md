**JasonBra1n CMS** is a custom-built, lightweight Content Management System designed to power [jasonbrain.com](https://jasonbrain.com) and serve as a centralized backend for various creative projects.


Hosted live at [CMS.jasonbrain.com](https://cms.jasonbrain.com).

## üöÄ Project Overview

This project marks a strategic pivot from static site generation to a dynamic PHP/SQL architecture. It separates content management from frontend presentation, allowing for easier updates and scalability across multiple sites.

### Key Features (In Development)
- **Centralized Dashboard**: Manage content for multiple frontend applications.
- **Custom PHP Framework**: Built from scratch with a focus on performance and understanding core principles.
- **Database Abstraction**: Efficient SQL interactions.
- **Secure Authentication**: Robust login system for administrative access.
- **API-First Design**: Serves content via JSON to frontend consumers.
